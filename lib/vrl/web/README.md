# Vector Remap Language Web

This directory houses the assets used to build the VRL web app.

## Prerequisites

* [npm]
* [Trunk]

## Setup

Install npm assets:

```bash
npm i
```

## Build

```bash
trunk build
```

## Serve

```bash
trunk serve
```

[npm]: https://npmjs.org
[trunk]: https://github.com/thedodd/trunk
